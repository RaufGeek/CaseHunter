# Case Hunter - React Frontend

Этот проект был переписан с обычного HTML/CSS/JS на React с использованием Vite в качестве сборщика.

## Технологии

- **React 19** - основная библиотека для UI
- **TypeScript** - типизированный JavaScript для лучшей разработки
- **Vite** - быстрый сборщик и dev-сервер
- **React Router DOM** - маршрутизация
- **SCSS** - препроцессор CSS
- **ESLint** - линтер для TypeScript/JavaScript

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── Header.tsx      # Шапка приложения
│   ├── Navigation.tsx  # Навигационное меню
│   ├── LoadingScreen.tsx # Экран загрузки
│   ├── CaseCard.tsx    # Карточка кейса
│   ├── InventoryItem.tsx # Элемент инвентаря
│   └── CasesModal.tsx  # Модальное окно кейсов
├── pages/              # Страницы приложения
│   ├── HomePage.tsx    # Главная страница
│   ├── UpgradePage.tsx # Страница апгрейда
│   ├── ReferralsPage.tsx # Страница рефералов
│   ├── LeaderboardPage.tsx # Таблица лидеров
│   └── ProfilePage.tsx # Профиль пользователя
├── styles/             # SCSS стили
│   ├── App.scss        # Основные стили
│   ├── components/     # Стили компонентов
│   ├── pages/          # Стили страниц
│   └── shared/         # Общие стили (кнопки, карточки, формы, модалки)
├── types/              # TypeScript типы
│   ├── index.ts        # Основные типы
│   └── global.d.ts     # Глобальные типы
├── assets/             # Статические ресурсы
├── utils/              # Утилиты
├── hooks/              # Кастомные React хуки
├── contexts/           # React контексты
└── services/           # Сервисы для работы с API
```

## Alias-ы для импортов

В проекте настроены удобные alias-ы:

- `@` - корень src/
- `@components` - папка компонентов
- `@pages` - папка страниц
- `@styles` - папка стилей
- `@assets` - папка ресурсов
- `@utils` - папка утилит
- `@hooks` - папка хуков
- `@contexts` - папка контекстов
- `@services` - папка сервисов

Пример использования:

```javascript
import Header from "@components/Header";
import { HomePage } from "@pages/HomePage";
import "@styles/App.scss";
```

## Установка и запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev-сервер:

```bash
npm run dev
```

3. Соберите проект для продакшена:

```bash
npm run build
```

4. Проверьте типы TypeScript:

```bash
npm run type-check
```

## Маршрутизация

Приложение использует React Router DOM со следующими маршрутами:

- `/` - Главная страница (Games)
- `/upgrade` - Страница апгрейда
- `/referrals` - Страница рефералов
- `/leaderboard` - Таблица лидеров
- `/profile` - Профиль пользователя

## Стили

Стили организованы в SCSS файлы с модульной структурой:

- `App.scss` - основные переменные и глобальные стили
- `components/` - стили для компонентов
- `pages/` - стили для страниц
- `shared/` - общие стили (кнопки, карточки, формы, модалки)
